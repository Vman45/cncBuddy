Text Control
To add text:
  1. Select the Text tool
  2. Move your mouse to the WorkArea
  3. Drag from the top left to the bottom right
  4. Double-Click inside the text shape to activate the TextEditor.

TextEditor
  • Control-EnterKey: Saves text and closes the editor
  • EscapeKey: Cancels edit-mode and closes the editor
