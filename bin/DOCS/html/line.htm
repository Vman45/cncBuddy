Line
To draw a line:
  1. Select the Line tool
  2. Move your mouse to the WorkArea
  3. Left-Click and Drag from the top left to the bottom right
  4. A new Line is created and the Properties Panel is updated

The Line Shape has two Sizing handles. The Start point is displayed in green and the End point in red:

  • To size a line, Left-Click and Drag the Start or End points.
  • To reposition a line, Left-Click and Drag within the border area of the line to the desired location.

Lines can be rotated by entering the desired angle in the RotateTo property. The angle is calculated from the center point between Start and End points...
