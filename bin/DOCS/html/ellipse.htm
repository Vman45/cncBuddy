Ellipse
To draw an ellipse:
  1. Select the Ellipse tool
  2. Drag from the top left to the bottom right
  3. Double-Click inside the text shape to activate the TextEditor.

The ellipse tool has four Control Handles:

  • Three rectangular ones control the size
  • One circular one (Left-Top) controls the rotation angle
  • Circle sizing handle (Bottom-Right)
