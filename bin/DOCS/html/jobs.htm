Jobs

The Job Panel defines the operations you wish to perform on cut parts including the required tools and materials.

Selecting an operation, tool, or material also displays the Details Panel below the planner/worksheet list. This allows you to maintain the properties associated with each selected item in the list.


Operations
  • Create a new Operation by clicking the Add Operation button on the main toolbar or...
  • Select an existing operation from the list
  • Drag a Tool from the ToolBox to the Details Panel
  • Select the Shapes to apply to this operation
  • Details Panel
    • Click AddItems to add the selected shapes will be added to the operation
    • Click DelItems to remove selected shapes from the operation

ToolBox

A global dictionary of all tools available to any worksheet operation. Adding a tool to your Toolbox makes it accessible to other worksheet operations.


Stock/Materials

A global dictionary of all materials available to any worksheet operation.
