Circle
To draw a circle:
  1. Select the Ellipse tool
  2. Move your mouse to the WorkArea
  3. Drag from the top left to the bottom right while holding down the Control key
  4. A new Circle is created and the Properties Panel is updated

The ellipse tool has four Control Handles:

  • Three rectangular ones control the size
  • One circular one (Left-Top) controls the rotation angle
  • Circle sizing handle (Bottom-Right)
