ShortCut Keys
Hints: Dock-able Windows
  • Use Ctrl+Tab to show window selector
  • Hover the mouse over the auto-hidden buttons to view window preview
  • Use Auto-Hide button to toggle the auto-hide mode of the window
  • Window (Right click mouse) to display the context menu

Menu Shortcut Keys
  • New = Ctrl-N
  • Open = Ctrl-O
  • Save = Ctrl-S
  • SaveAs = Ctrl-Shift
  • Exit = Alt-F4
  • Undo = Ctrl-Z
  • Redo = Ctrl-Y
  • SelectAll = Ctrl-A
  • UnSelectAll = Ctrl-Shift
  • BringToFront = Ctrl-F
  • SendToBack = Ctrl-Shift
  • Delete = Delete
  • Cut = Ctrl-X
  • Copy = Ctrl-C
  • Paste = Ctrl-V
  • Help = Ctrl-F1
  • Operation_Pocket = Alt-P
  • Operation_Inside = Alt-I
  • Operation_Outside = Alt-O

WorkArea Shortcut Keys
  • Select Multiple = Alt-Click
  • Zoom In/Out = Ctrl-MouseWheel
