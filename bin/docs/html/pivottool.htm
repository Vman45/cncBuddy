Pivot Tool
  1. Select a Shape/CutPart to pivot.
  2. Select the Pivot tool
  3. Click any where on Material to anchor the Pivot tool. This becomes the center pivot point. Pivot anchor can be dragged to a new position when "arrow" icon is visible.
  4. While the "rotate" icon is visible, click on the Shape/CutPart anchor point. Auto-Snap will position to the closest anchor point (TopLeft or Center).
  5. ClickNDrag on Material to pivot angle
  6. Use the Escape Key to exit Pivot mode
Shortcut Keys

When hovering on the Shape's outline:

  • Alt-Click: Deletes a shape handle
  • Ctrl-Click: Inserts a new shape handle
