FileOpen
  • If the document is not already loaded:
    • Preprocess special formats (Inkscape,SVG,Image,etc)
    • Inkscape with GCodetool properties are converted to cncBuddy formats
    • Finally it's added to the Recent Files menu list
  • If the document is already loaded, it's WorkArea tab is selected
